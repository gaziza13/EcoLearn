# Contributing to EcoLearn

Спасибо, что хотите помочь!

## Как начать
1. Сделайте fork репозитория
2. Создайте новую ветку:  
   `git checkout -b feature-name`
3. Внесите изменения
4. Пройдите lint/tests (если есть)
5. Отправьте Pull Request

## Требования к коду
- Пишите простой и понятный код
- Коммиты должны быть в формате:  
  `feat: ...`, `fix: ...`, `docs: ...`
- Добавляйте комментарии к сложным частям

## Как создавать задачи
Используйте GitHub Issues:
- Bug report
- Feature request
- Documentation request

## Code of Conduct
Все участники обязаны соблюдать `CODE_OF_CONDUCT.md`.
