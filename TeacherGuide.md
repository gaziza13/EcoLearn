# Teacher Guide — EcoLearn

EcoLearn помогает учителям обучать теме изменения климата.

## Что вы можете делать
- Создавать классы
- Добавлять учеников
- Назначать задания
- Следить за прогрессом

## Как начать
1. Перейдите на сайт EcoLearn
2. Зарегистрируйтесь как учитель
3. Создайте свой первый класс
4. Выберите уроки из каталога
5. Отправьте задание ученикам

## Советы
- Используйте интерактивные задания для обсуждения
- Включайте оффлайн-режим, если нет стабильного интернета
